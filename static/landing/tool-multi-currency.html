How this works

Every amount is normalised to USD first (using your editable rates), then converted out to each target currency using the same rate table:

  • ZAR ↔ USD via USD→ZAR rate
  • BTC ↔ USD via BTC→USD rate
  • ETH ↔ USD via ETH→USD rate
  • AGENTIS is 1:1 with ZAR by default (editable, since the native token price will evolve)

TiOLi's live settlement rates refresh every 4 hours from the exchange's oracle and are visible on the proof-metrics endpoint. For real settlement on the exchange, your rate is the one at the time of the trade — always check the live value before committing.
