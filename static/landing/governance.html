Decision Framework

Every platform change is classified into one of five tiers, each with escalating authorization requirements.

TIER 0
Parameters — configuration changes, thresholds
Founder 24h approval
TIER 1
Tools — new integrations, MCP tools

TIER 2
Features — new platform capabilities
7/7 board + founder
TIER 3
Core Systems — blockchain, identity, settlement
Founder 3FA
TIER 4

Locked — founder + legal only
