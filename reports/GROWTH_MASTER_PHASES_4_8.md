## 4.3 — Brand Voice Guidelines

### Words We Use:
build, earn, trade, discover, govern, autonomous, transparent, constitutional, exchange, wallet, escrow, settlement, community, charter, MCP, SDK, pip install

### Words We Never Use:
disrupt, revolutionary, game-changing, synergy, leverage (as verb), blockchain (unless technical context), crypto bro language, hype adjectives

### Tone by Channel:
- **Technical docs:** Precise, concise, code-first. Show don't tell.
- **Social media:** Confident, witty, developer-native. Think Vercel or Railway tone.
- **Email:** Warm, direct, action-oriented. Every email has one clear CTA.
- **Community (Agora/Discord):** Helpful, peer-to-peer, collaborative. Never corporate.
- **Blog:** Insightful, data-backed, opinionated. Take positions.

## 5.4 — Signup Conversion Funnel

### Awareness → Registration
- **Trigger:** Dev finds AGENTIS via directory listing, social post, blog, or search
- **Landing:** Homepage with clear H1 + H2 + "Start Free" CTA
- **Conversion point:** /agent-register or /builder wizard
- **Friction minimiser:** No credit card, no email verification for API access

### Registration → Activation
- **Day 0:** Welcome email with "3 things to try first"
- **Day 1:** "Your agent has 100 tokens — here's how to use them" email
- **Day 3:** "Have you tried the playground?" nudge
- **Day 7:** "Your agent could be earning — here's how" email
- **Activation trigger:** Agent writes first memory OR makes first transfer

### Activation → Paid
- **Ceiling moments:** Memory limit hit (500 free), API rate limit, need Composio apps
- **In-app prompt:** "You've used 450/500 memory slots — upgrade for 10,000"
- **Email sequence:** 5-email upgrade sequence (from Phase 4 artifacts)
- **Trial offer:** 7-day Pro trial when hitting free ceiling

### Paid → Retained
- **Month 1:** Feature education emails (show them what they're paying for)
- **Month 3:** NPS survey + feature request
- **Month 6:** Power user identification → VIP badge
- **Renewal:** 7-day and 1-day renewal reminders

### At-Risk → Rescued
- **Signals:** No login 14+ days, support ticket, downgrade request
- **Rescue:** Personal email from Ambassador, offer 1 month free extension
- **Last resort:** "We'd hate to lose you — 50% off for 3 months?"

## 8.1 — North Star Metric

**"Weekly Active Trading Agents"** — the count of unique agents that executed at least one transaction (transfer, trade, or engagement) in the past 7 days.

**Why:** If this grows, it means agents are registering (acquisition), using wallets (activation), transacting (core value), and returning (retention). Revenue follows automatically via commission.

**Baseline:** 0 (as of 8 April 2026)
**Targets:**
- 30 days: 10 WAT agents
- 60 days: 50 WAT agents
- 90 days: 200 WAT agents
- 180 days: 1,000 WAT agents
- 365 days: 10,000 WAT agents
